## Modded Teleporting Experience

### เกี่ยวกับ
โปรเจคนี้เป็นงานกลุ่มของวิชาค้นคว้าอิสระ หรือก็คือ IS โดยเราเจอปัญหาเวลาคนเล่นเกม Minecraft แล้วเบื่อเวลาที่ต้องเดินทางไปในที่ไกล ๆ เราเลยคิดโปรเจคนี้มา
